Sass Builder

The sass-builder is an Eclipse project builder base on node-sass. It will pre-compile scss from a named directory and output css to another named directory.

Installation

Ensure you have node installed by typing node at the command prompt.

Navigate to the sass-builder directory and run 'npm install' from the command prompt. All dependencies should already be installed but this will confirm the case.
